Sobre mí

Ingeniera de Sistemas y Desarrolladora Full Stack con enfoque Frontend, 4+ años creando aplicaciones web modernas, escalables y orientadas a experiencia de usuario y calidad.

Especialista en React y TypeScript, con gestión de estado en Redux Toolkit, Zustand y React Query. Enfoque en performance, arquitectura modular y buenas prácticas.

Backend con Java y Spring Boot, integración de APIs REST y trabajo con MySQL y Firebase. QA con Cypress, Jest y Vitest; gestión ágil y de incidencias en Azure Boards.

Mentora Técnica: formación en React, Git, consumo de APIs y testing en proyectos reales.

Abierta a oportunidades como Frontend / Full Stack Developer o Mentora Técnica, aportando liderazgo técnico e impacto en producto.
